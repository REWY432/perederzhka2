# DogStay Manager

## Как запустить приложение

Этот проект написан на React + TypeScript и использует Vite для сборки. Браузер не может запустить его файлы напрямую.

1.  **Установите зависимости**:
    ```bash
    npm install
    ```

2.  **Запустите сервер разработки**:
    ```bash
    npm run dev
    ```

3.  Откройте ссылку в браузере (обычно `http://localhost:5173`).

## Настройка Google Таблицы (Backend)

Приложение хранит данные в Google Таблице через Apps Script.

1.  Создайте новую [Google Таблицу](https://sheets.google.com).
2.  В меню выберите **Расширения** (Extensions) -> **Apps Script**.
3.  Очистите файл `Code.gs` и вставьте содержимое файла `backend/Code.js` из этого проекта.
4.  Нажмите кнопку **Сохранить** (дискета).
5.  Запустите функцию `setup` один раз (выберите `setup` в выпадающем списке сверху и нажмите **Выполнить/Run**). Это создаст нужные колонки в таблице.
    *   Вам потребуется дать разрешение скрипту на доступ к таблице.
6.  Нажмите синюю кнопку **Начать развертывание** (Deploy) -> **Новое развертывание** (New deployment).
7.  В окне выберите:
    *   **Выберите тип**: Веб-приложение (Web app).
    *   **Описание**: Любое (например, v1).
    *   **Выполнять как**: Я (Me).
    *   **У кого есть доступ**: **Все (Anyone)**. (Это важно, иначе приложение не сможет отправлять запросы).
8.  Нажмите **Начать развертывание** (Deploy).
9.  Скопируйте полученный **URL веб-приложения**.
10. Откройте приложение DogStay, нажмите **Настройки** и вставьте ссылку в поле "Google Sheets Web App URL".
